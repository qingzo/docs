# JadeView VolDP SDK

The JadeView VolDP module is used to call JadeView's window, WebView, IPC, tray, notification, dialog, and system capabilities within a VolDP project.

## Installation

1. Download the module installation package
2. Install it into the VolDP module directory
3. Reference `火山.JadeView` in your project
