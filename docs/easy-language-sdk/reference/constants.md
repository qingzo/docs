## 使用建议

1. 开发优先使用 SDK 内置公开常量，禁止手写原始字符串 / 数字，统一代码规范，减少拼写错误；
2. 生命周期、交互事件注册必须使用对应事件常量，不可手写底层事件字符串；
3. 窗口主题默认推荐 `#主题_自动`，适配用户系统明暗偏好。

## 注意事项

1. 常量底层为固定字符串 / 数字，直接手写原值可运行，但不利于后期维护；
2. 切换窗口主题会触发页面重新渲染，短暂闪烁属于正常现象；
3. Win7 / 旧版 Win10 不支持 Mica、Acrylic 透明材质，对应常量自动降级普通背景；
4. 事件常量仅用于事件注册接口，不可用作 IPC 通讯通道名称；
5. JAPK 相关错误码仅商用加密包场景返回，本地明文资源无相关错误。
