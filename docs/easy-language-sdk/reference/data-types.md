# 数据类型

JadeView 易语言 SDK 提供全套公开自定义数据类型，用于窗口、Web 内核、弹窗、托盘、通知、资源配置等场景，统一参数格式，规范传参逻辑。所有中文文本成员赋值前，必须使用 `GBK文本到UTF8文本()` 编码转换，防止乱码。

## 一、顶层入口 JadeView（全局单例）

```
.数据类型 JadeView, 公开, JadeView 全局入口类，导入ec后自动生成全局单例无需手动创建
    .成员 App, _JadeViewApp
    .成员 窗口, _JadeView_窗口
    .成员 对话框, _JadeView_对话框
    .成员 Yaml配置项, _JadeView_YAML
    .成员 托盘, _JadeView_托盘
    .成员 右键菜单, _JadeView_右键菜单
    .成员 页面, _JadeView_页面
    .成员 通讯, _JadeView_IPC
    .成员 协议服务, _JadeView_自定义协议服务
    .成员 系统, _JadeView_系统集成
    .成员 通知, _JadeView_系统通知
    .成员 文本, _JadeView_文本处理
    .成员 剪切板, _JadeView_剪切板
    .成员 JAPK, _japk
    .成员 时间, _jadeview_时间
```

### 说明

SDK 顶层聚合类型，拆分 15 个功能子程序集，覆盖全部业务 API，所有操作均通过 `JadeView.xxx.子程序()` 调用。

## 八、通用注意事项

1. **中文编码强制规则**：所有中文文本型成员赋值前，必须调用 `GBK文本到UTF8文本()`，内核仅接收 UTF8 字符串，不转换会出现乱码。
2. **互斥属性限制**：`透明背景=真` 时 `背景颜色` 配置失效，二者不能同时生效。
3. **常量优先原则**：主题、边框、弹窗类型、对话框特性、托盘类型全部使用 SDK 公开常量，禁止手写原始字符串 / 数字。
4. **窗口坐标规则**：X/Y 坐标赋值 `-1` 代表屏幕自动居中；数值为窗口左上角像素坐标。
5. **无痕模式副作用**：开启无痕后 Web 缓存、Cookie、本地存储不会持久化，关闭页面全部销毁。
6. **CORS 白名单**：本地 jade:// 协议页面无需配置，第三方 http 页面跨域必须添加域名白名单。
7. **托盘 Key 规范**：菜单项 Key 仅允许英文、数字、下划线，禁止中文，否则事件匹配失效。
8. **资源释放**：窗口、通知、托盘无需手动释放，程序执行 `JadeView.App.退出()` 统一回收全部句柄。

## 九、总结

SDK 所有业务场景均配套专用自定义数据类型，参数结构标准化，替代零散独立传参。开发时优先使用对应结构体完整配置窗口、弹窗、托盘、通知，配合全局常量与内置编码工具，大幅减少报错、提升代码可读性与可维护性。
